## Peering

For developers interested in co-location we can offer the following:

- Publicly routable address space (at least one /24)
- 24x7 NOC contact capable of resolving technical/routing issues (response time t.b.d.).
- Presence at one or more of the datacenter/peering interconnection facilities listed (primarily
  OVHCloud).

## API Endpoints

| **RPC**                      | **Endpoint**                                                                                |
| ---------------------------- | ------------------------------------------------------------------------------------------- |
| Status                       | `/eth/v1/builder/status`                                                                    |
| RegisterValidator            | `/eth/v1/builder/validators`                                                                |
| GetHeader                    | `/eth/v1/builder/header/{slot:[0-9]+}/{parent_hash:0x[a-fA-F0-9]+}/{pubkey:0x[a-fA-F0-9]+}` |
